# Passwd Manager

A simple password manager GUI built with [Dear PyGui](https://github.com/hoffstadt/DearPyGui).

### There is no encryption now, security will be added in 1 - 10 years

## Features
- Store site credentials (name, login, mail, password, notes)
- Add, edit, and delete resources
- Password visibility toggle
- SQLite database backend

## Requirements
pip install dearpygui

shell
Copy code

## Run
python main.py
